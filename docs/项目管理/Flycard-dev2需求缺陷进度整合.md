## 8. 已落地到当前系统的导入脚本

- `server-nestjs/prisma/seed-flycard-dev2-project.ts`

1. Flycard 项目基础信息。

3. dev2 迭代。
4. dev2 里程碑。

为避免未确认的数据写入，我没有自动执行数据库导入。确认要写入当前数据库后，可在项目根目录执行：

```bash
pnpm --filter server-nestjs exec ts-node prisma/seed-flycard-dev2-project.ts
```

执行前建议先备份数据库，并确认 `.env` 中 `DATABASE_URL` 指向的是目标环境。
