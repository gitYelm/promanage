## 🆕 最近更新

### 2026-01-08
- 富文本编辑器支持表格（可配置行列数）、图片上传、视频上传
- 上传大小限制可通过「参数配置」动态调整
- 修复多个弹窗内容过长时无法滚动的问题

### 上传配置参数

| 参数键 | 说明 | 默认值 |
|--------|------|--------|
| `sys.upload.editor.imageMaxSize` | 编辑器图片大小限制 | 5MB |
| `sys.upload.editor.videoMaxSize` | 编辑器视频大小限制 | 50MB |
| `sys.upload.avatar.maxSize` | 头像大小限制 | 2MB |
| `sys.upload.system.maxSize` | 系统文件大小限制 | 2MB |
